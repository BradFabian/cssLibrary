Features

  • Very small file!
  • Easy to use. No config required!
  • Works in all browsers

How to Use

Check out the readme at the github repo. Then in your HTML include the CDN LINK:

<link rel="stylesheet" href="toolbox.css">

Set the div position to "relative" then add a toolbox class to your span

<div class="tooltip-container"> Accept <span class="toolbox-accept"> Accepted </span> </div>
